# cryptnote
Simple Notepad in Python (Tkinter) with the option to encrypt files using a simple random Caesar Cipher

## Requirements
1) Python 3+
2) various pre-installed python modules (tkinter, pickle, random, os)
3) pip if required to install the modules (`pip install <module_name>`)

## To run
`py main.py` or `python main.py` or `python3 main.py`

for bruteforce test, run `py Cipher_Crack.py` then give the name of the encrypted file to crack when prompted.
